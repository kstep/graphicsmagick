The GraphicsMagick mailing list are a low noise and subject oriented. The subject is the discussion of GraphicsMagick software and its use. Although the lists are unmoderated, do not post off-topic or test messages to the lists. Off-topic messages will result in the offender being silently removed from the lists and prevented from rejoining.

Subscribe to a list
Send e-mail with the subject
subscribe
to one or more these lists:
magick-users-request@imagemagick.org
magick-developers-request@imagemagick.org
magick-bugs-request@imagemagick.org
magick-announce-request@imagemagick.org
The purpose of the lists is as follows:
List
Description
user
Questions and postings pertaining to the usage of GraphicsMagick regardless of the interface. This includes the command-line utilities, as well as the Perl, C++, and C APIs. Usage questions are like "How do I use GraphicsMagick to create drop shadows?".
developer
Questions and postings pertaining to the development of GraphicsMagick, feature enhancements, and GraphicsMagick internals. GraphicsMagick source code and algorithms are discussed here. Usage questions which are too arcane for the normal user list should also be posted here.
bugs
Post any defects you find in the GraphicsMagick software here. The GraphicsMagick maintainers carefully monitor this list and will work diligently to repair any reported problems. When posting a bug report, please specify both the GraphicsMagick version and operating system you are using.
announce
Announcements pertaining to GraphicsMagick, or GraphicsMagick related software. This list is moderated and will not be used for any discussion.
