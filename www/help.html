A number of individuals volunteer their time to develop and maintain GraphicsMagick. However, we can't do it alone. We are looking for other volunteers to help make GraphicsMagick even better. Here is a list of tasks you can do to contribute to this effort:

  • Write a book about GraphicsMagick and how to utilize it's utilities and API.

  • Write a program on the Macintosh to display images on the screen using the GraphicsMagick API.

  • Write a graphical user interface to configure the delegates.mgk file.

  • Write an interface between Tcl/Tk and libMagick.

  • Write a program that calls the GraphicsMagick ReadImage routine and displays the image in a native window on the Macintosh or Windows NT.

  • Write a special effect routine such as page curl or image mosaic.

  • Improve the GraphicsMagick documentation. The most useful improvement would be a HTML section that describes how to perform some of the more useful GraphicsMagick functions with actual images to illustrate the results.

  • Send bug reports or any suggestions you have for improving GraphicsMagick.
