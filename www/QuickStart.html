GraphicsMagick Binary Distributions

This page documents how to download and install the binary distribution packages built by GraphicsMagick Studio, as found in the GraphicsMagick 'binaries' subdirectory of GraphicsMagick ftp sites. Those who prefer to build GraphicsMagick from source code should refer to the source code installation guide.

Unix/Linux/Darwin/Cygwin Install

Download the appropriate binary from ftp://ftp.imagemagick.com/pub/GraphicsMagick/binaries or a mirror site.

The available Unixish binaries are shown in the following table:

Package Name

Description

GraphicsMagick-alphaev6-unknown-linux-gnu.tar.gz

Linux on Compaq Alpha processor

GraphicsMagick-i686-pc-cygwin.tar.gz

Cygwin environment for Microsoft Windows

GraphicsMagick-i686-pc-linux-gnu.tar.gz

Red-Hat Linux for Pentium II

GraphicsMagick-mips-sgi-irix6.5.tar.gz

SGI IRIX

GraphicsMagick-powerpc-apple-darwin5.5.tar.gz

Mac OS X "Darwin"

GraphicsMagick-sparc-sun-solaris2.9.tar.gz

Solaris SPARC (requires 64-bit kernel)

Create (or choose) a directory to install the package into and cd to that directory, for example:

mkdir $HOME/GraphicsMagick

cd $HOME/GraphicsMagick

Extract the contents of the package using the command

gzip -dc /path/to/package.tar.gz | tar -xf -

If the 'bin' subdirectory of the extracted package is not already in your executable search path, add it to your PATH environment variable. For example

export PATH="$HOME/GraphicsMagick:PATH"

Test GraphicsMagick to see if it is working

convert LOGO: logo.miff

display logo.miff

GraphicsMagick tries to automatically find its components, but occasionally it may fail. If an error is displayed that a file can not be found, you may need to set the MAGICK_HOME environment variable to specify path where you extracted the GraphicsMagick files. For example

export MAGICK_HOME="$HOME/GraphicsMagick"

Additional Information

Environment Variables:

In addition to the MAGICK_HOME environment variable described above, you may find these variables useful:

Variable Name

Description

MAGICK_HOME

Path to GraphicsMagick installation directory (directory above 'bin' directory).

Megabytes of memory available to pixel cache

MAGICK_FONT_PATH

Directory containing Truetype and/or Postscript Type 1 font files.

TMPDIR

Path to store temporary files

LD_LIBRARY_PATH

Path to find libMagick.so and other shared libraries

Configuration Files

GraphicsMagick depends on a number of external configuration files which include colors.mgk, delegates.mgk, magic.mgk, type.mgk, and modules.mgk. GraphicsMagick searches for configuration files in the following order, and will use the first file found:

<client path>/../lib/GraphicsMagick/

$MAGICK_HOME/lib/GraphicsMagick/

$HOME/.magick/

<current directory>/

Font Files

GraphicsMagick is able to load raw TrueType and Postscript font files. It searches for these files in the following order and uses the first match found:

<current directory>/

$MAGICK_FONT_PATH/

$MAGICK_HOME/

$HOME/.magick/

Module Files

GraphicsMagick's file format support is usually provided in the form of loadable modules. It searches for loadable modules in the following order and it uses the first match found:

<client path>/../lib/GraphicsMagick/modules/coders/

$MAGICK_HOME/lib/GraphicsMagick/modules/coders/

$HOME/.magick/

<current directory>/


XP, Win2K, WinME, NT, Win98, Win95 Install

Download

ftp://ftp.imagemagick.com/pub/GraphicsMagick/binaries/GraphicsMagick-i686-pc-windows.exe or ftp://ftp.imagemagick.com/pub/GraphicsMagick/binaries/GraphicsMagick-i686-Q8-windows.exe

and execute it from the Windows Command Shell, double-click from Windows Explorer, or just "open" it from your browser, to start the installation program.

Once GraphicsMagick is installed, Select Start->Programs->Command Prompt. In the Command Prompt window type

convert logo: logo.miff

imdisplay logo.miff

If you have an X11 server and would like to test GraphicsMagick with it, from the Command Prompt window type

set DISPLAY=:0

display

Macintosh MacOS 9 Installation
Note, if you are using MacOS-X ("Darwin") please refer to the Unix installation instructions.

Download

ftp://ftp.imagemagick.com/pub/GraphicsMagick/binaries/GraphicsMagick-powerpc-apple-mac9.2.sit

and double click on it to extract the distribution. Move to the GraphicsMagick folder and double click on the convert program icon. Next, type

-verbose logo: logo.jpg

in the command window and choose File->Quit to exit.

VMS Installation
There is no binary package available for VMS.

Type

unzip GraphicsMagick-5.5.1.zip

set default [.imagemagick]

@make

identify -verbose logo:

Image manipulation software that works like magic.
